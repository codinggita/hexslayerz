# Roadmap

## Future Enhancements

- **Live AI API Integration**: Replace the `StubProvider` with an actual network client capable of streaming chat completions from Gemini, Claude, or OpenAI.
- **Advanced Export Formats**: Support exporting Checkpoints to Markdown, PDF, or Notion.
- **Cloud Sync**: Optional opt-in to sync checkpoints across Chrome profiles via `chrome.storage.sync` or external OAuth.
- **Full Text Search Index**: Integrate a lightweight search library (e.g., MiniSearch) for high-performance fuzzy matching on massive checkpoint libraries.
